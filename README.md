# FilesCenter

## Free Online PDF, Image & Developer Tools

FilesCenter is a collection of free, privacy-first online tools. All processing happens directly in your browser, so your files never leave your device.

## Features

- **100% Free**: No hidden costs or premium tiers.
- **Privacy First**: Files are processed client-side. No server uploads.
- **No Registration**: All tools are available instantly without an account.
- **50+ Tools**: PDF, Image, Text, Developer, Calculator, and Converter tools.
- **Modern UI**: Clean, responsive design with dark and light themes.

## Comprehensive Tool List

### PDF Tools

- **Merge PDF**: Combine multiple PDF files into one.
- **Split PDF**: Extract specific pages or split a PDF into multiple files.
- **Compress PDF**: Reduce PDF file size while maintaining quality.
- **PDF to Image**: Convert PDF pages to JPG or PNG images.
- **Rotate PDF**: Change the orientation of PDF pages.
- **Watermark PDF**: Add text or image watermarks to your PDF.
- **Protect PDF**: Add password protection to your PDF files.
- **Unlock PDF**: Remove passwords and restrictions from PDFs.

### Image Tools

- **Compress Image**: Optimize images for web and storage.
- **Resize Image**: Change image dimensions.
- **Crop Image**: Trim edges or focus on specific parts of an image.
- **Convert Format**: Convert between JPG, PNG, WebP, and more.
- **Image to PDF**: Create a PDF from one or more images.
- **Base64 Converter**: Convert images to Base64 strings.
- **Flip/Rotate**: Mirror or rotate images as needed.

### Text Tools

- **Word Counter**: Count words, characters, and reading time.
- **Case Converter**: Toggle between UPPERCASE, lowercase, Title Case, etc.
- **Lorem Generator**: Generate placeholder text for designs.
- **Diff Checker**: Compare two pieces of text for differences.
- **Duplicate Remover**: Clean up lists by removing repeated lines.
- **Markdown Preview**: Real-time rendering of Markdown text.
- **Slug Generator**: Convert titles into URL-friendly slugs.

### Developer Tools

- **JSON Formatter**: Clean up and validate JSON data.
- **Base64 Encoder/Decoder**: Securely handle Base64 data.
- **UUID Generator**: Create unique identifiers instantly.
- **Hash Generator**: Generate MD5, SHA-1, SHA-256 hashes.
- **URL Encoder/Decoder**: Format strings for URL safety.
- **Regex Tester**: Test regular expressions in real-time.
- **JWT Decoder**: Decode and inspect JSON Web Tokens.
- **Epoch Converter**: Convert timestamps to human-readable dates.

### Calculators & Converters

- **Financial**: Loan and Tip calculators.
- **Health**: BMI and Age calculators.
- **Unit Converter**: Mass, length, volume, and temperature.
- **Currency Converter**: Real-time exchange rates.
- **Timezone Converter**: Sync times across different regions.
- **Color Picker**: Pick and convert between HEX, RGB, and HSL.

### Generators

- **QR Code Generator**: Create QR codes for URLs and text.
- **Password Generator**: Build secure, custom passwords.

## Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **PDF Processing**: `pdf-lib`, `pdfjs-dist`
- **Image Processing**: Canvas API, `react-image-crop`
- **Analytics**: Vercel Analytics

## Getting Started

```bash
npm run dev
# or
yarn dev
# or

```

Open [http://localhost:3000](http://localhost:3000) for the local version.
